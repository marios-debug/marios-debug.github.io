# Marios Portfolio (Static Site)

## How to open locally
- Open `index.html` in a browser.
- No build tools required.

## Where to edit text
- Home: `index.html`
- Reflection (Part 1): `reflection.html`
- Checklist (Part 2): `checklist.html`
- Downloads: `downloads.html`
- Styling: `assets/css/style.css`
- Animations: `assets/js/main.js`

## Where to place PDFs
Put files inside `assets/downloads/` and keep the filenames used on the Downloads page:
- `cv-marios-en.pdf`
- `cv-marios-fr.pdf`
- `cv-marios-de.pdf` (optional)
- `certificates.zip`
- `diplomas.pdf`
